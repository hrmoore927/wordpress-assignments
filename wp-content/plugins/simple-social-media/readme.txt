== Description ==

Simple Social Media lets you add rounded social media icons to your WordPress page, post or text-widget.



**FEATURES**

1. More than 10 of the most popular social media platforms supported.

2. You can choose your custom color for the icons to make them fit your theme / design.

3. Simple to use thanks to the generated shortcode to output your chosen icons anywhere you want on your page.



== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress

3. Go to the admin panel "Social Media"

4. Enter the URLs of your social media accounts into the inputfields

5. Choose a color for the icons

6. Save changes

7. Display icons by pasting the shortcode [social_media] to a page, post or text-widget
